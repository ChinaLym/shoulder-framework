# 开始学习 Shoulder - 应用监控 ！

下载本项目

```
git clone https://gitee.com/ChinaLym/shoulder-framework-demo
```

打开 `demo4` 工程，将其在本地运行（默认8080端口）

打开 [http://localhost:8080/actuator/prometheus](http://localhost:8080/actuator/prometheus) ,可以看到其指标（关键字 demo4_thread_pool ）
